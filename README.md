## How to Run Locally

1. Clone the repository:
    ```bash
    git clone https://github.com/votre-nom-utilisateur/star-wars-animation.git
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the development server:
    ```bash
    npm run dev
    ```

4. Open `http://localhost:3000` in your web browser.
